# LAN Server (Experimental)

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   # or: node server.js
   ```

3. The other player connects using your local IP + port (default 8765).

Example:
- You run the server on `192.168.1.42`
- Other player enters `192.168.1.42:8765` in the game UI

This is the absolute simplest possible multiplayer server for testing purposes.
